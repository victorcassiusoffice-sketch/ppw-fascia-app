## If Sub-Chat 3 used different filenames

Add a mapping in `src/data.js` under the new export `CHAIN_OVERLAY_FILES`:

```js
export const CHAIN_OVERLAY_FILES = {
  superficial_back_line:  { front: 'sbl_front.png', back: 'sbl_back.png' },
  // ...
};
```

The BodyMap component reads from this map and falls back to the default `<chain>_<view>.png` pattern if no entry exists.

## Specs

- 600 × 1200 PNG with transparency (matches the silhouette canvas viewBox)
- Gold (`#f5b845`) glow / outline showing the chain path
- File size: ~1 MB each → ~16 MB total for the set
